->> Sellzy – Multipurpose eCommerce Website

A responsive, multipurpose eCommerce website built with HTML, CSS, and JavaScript. It includes multiple homepage layouts, shop and product pages, cart, wishlist, checkout, authentication, and blog pages.

->> Features

- Multiple homepage variations (`index.html`, `index-2.html` to `index-5.html`)
- Shop page with product listing (`shop.html`)
- Single product & product detail view (`single-product.html`, `view-product.html`)
- Shopping cart (`cart.html`)
- Wishlist (`wishlist.html`)
- Checkout flow (`checkout.html`)
- User authentication (`login.html`)
- My account dashboard (`my-account.html`)
- Seller listing (`sellers.html`) and add product form (`add-product.html`)
- Blog (`blog.html`), About Us (`about-us.html`), Contact (`contact.html`)
- Mega menu navigation (`mega-menu.html`)
- Fully responsive design

->> Tech Stack

- **HTML5** – page structure
- **CSS3** – styling (`assets/css/style.css`)
- **JavaScript (Vanilla + jQuery)** – interactivity
  - `auth.js` – login/authentication logic
  - `cart.js` – shopping cart logic
  - `products.js` – product listing/handling
  - `wishlist.js` – wishlist logic
  - `main.js` – general site scripts
