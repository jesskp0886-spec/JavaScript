# 🚀 TypeScript Programming Challenges Collection

A comprehensive collection of **20 beginner-to-intermediate TypeScript programs** designed to strengthen problem-solving skills, logical thinking, and understanding of core programming concepts.

This project covers everything from basic loops and conditions to number manipulation algorithms such as **Palindrome**, **Armstrong Number**, **Factorial**, and **Multiplication Tables**.

## ⚙️ Getting Started

### 1️⃣ Install TypeScript

```bash
npm install -g typescript
```

### 2️⃣ Compile the TypeScript File

```bash
tsc index.ts
```

### 3️⃣ Run the Program

```bash
node index.js
```
