# 🌦️ Weather Forecast Application

A modern and responsive **Weather Forecast Application** built using **HTML, CSS, and JavaScript**. The application allows users to search for any city and view the current weather details along with a 5-day weather forecast using the OpenWeatherMap API.

---

## ✨ Features

- 🔍 Search weather by city name
- 🌡️ Current temperature in Celsius
- 🌤️ Weather condition with dynamic icon
- 📍 City name and country
- 🌎 Latitude and Longitude
- 🤗 Feels Like temperature
- 💧 Humidity
- 🌬️ Wind speed
- 📅 Current date display
- 📆 5-Day weather forecast
- ⌨️ Search using Enter key
- ⏳ Loading animation
- ❌ Error handling for invalid city
- 📱 Responsive design

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- OpenWeatherMap API
- Font Awesome
- Google Fonts (Inter)
