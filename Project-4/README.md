🧠 Aesthetic Lifestyle & Philosophy Quiz

A clean, modern, and responsive quiz application built using HTML, CSS, and JavaScript. The project demonstrates core front-end development concepts including DOM manipulation, event handling, timers, responsive layouts, and dynamic UI updates.

---

✨ Features

- Interactive multiple-choice quiz
- Countdown timer for each question
- Dynamic progress bar
- Question counter
- Automatic answer validation
- Correct and incorrect answer highlighting
- Automatic navigation to the next question
- Final score calculation
- Restart quiz functionality
- Fully responsive design

---

🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Google Fonts (Poppins)
- Font Awesome

🔄 Project Workflow

1. User starts the quiz.
2. Questions load dynamically.
3. Timer starts for each question.
4. User selects an answer.
5. Answer is validated instantly.
6. Correct answer is highlighted.
7. Next question loads automatically.
8. Final score is displayed after the last question.
